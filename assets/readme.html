Suite à une modification de chez google l'export takeout est maintenant encodé en BASE64. Donc on ne s'en sert plus, tout se fait dans _dynnamicNoteHtml

  1. Télécharger le drive en HTML
  2. Le mettre dans le dossier /asset, aller dessus, et dans la console copier le JSON généré
  3. Coller le json dans un nouveau dossier lié au livre (final.json)
  1. Télécharger le drive en HTML : supprimer toute la partie haute en doublon et ne garder que le texte après "Toutes vos annoations" - ce fichier est utile car contrairement au takeout les notes ici ne sont pas tronquées
  2. Aller sur indexDrive.html et copier/coller le JSON dans drive.json : Le nombre de donnée dans le json doit dorrespondre avec les notes (fonction _dynnamicNoteHtml)
  3. Récupérer le takeout en Json : mettre dans le fichier takeout.json (enlever le haut et le bas du json pour qu'il soit correctement lu) - ce fichier est utile pour prendre la couleur et les notes avant et après
  4. Note : il se peut que dans le takeout.json il y ait des problèmes :
    1. problèmes d'espace : dans ce cas remplacer l'espace google " " par un espace normal
    2. Remplacer les "\n" par ""
  5. Pour terminer : Copier / coller le json qui sort dans la console de index.html dans final.json (normalement le nombre de data devrait être le même que ceux du drive.json)
  6. La sortie finale se trouve dans un fichier html que vous aurez créée avec une classe html final
